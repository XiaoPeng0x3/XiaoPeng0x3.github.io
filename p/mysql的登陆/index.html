Mysql的登陆

2025-03-04
阅读时长: 1 分钟

启动服务

在win下可以使用net启动mysql服务。

  • 首先打开具有管理员权限的cmd窗口，否则将会被禁止访问

  • 在窗口中输入

    1

    net start mysql

    然后就可以等待服务开启

  • 如果想要关闭服务，只需要输入

    1

    net stop mysql

登录服务

在mysql中可以指定用户的登录，其命令格式如下‘

1

mysql -u root -p password

-u参数后面接的是登录时的用户名

如果想要登出mysql，

1

quit;

密码相关

修改密码

假设已经知道原来的密码，比如原来的密码是123，想要把密码修改为123456。

  1. 登录mysql

    1

    mysql -u root -p 123

  2. 修改表

    1

    SET PASSWORD FOR 'root'@'localhost' = '123456';

  3. 刷新权限

    1
2

    FLUSH PRIVILEGES;
EXIT;

  4. 登出

  5. 可以进行验证

忘记密码

  1. 关闭所有正在运行的mysql服务

    1

    net stop mysql

  2. 以跳过授权的方式登录

    1

    mysqld --skip-grant-tables --skip-networking

  3. 直接登录

    1

    mysql -u root

    然后进行修改

    1
2
3

    ALTER USER 'root'@'localhost' IDENTIFIED BY '123456';
FLUSH PRIVILEGES;
EXIT;

  4. 重启mysql

    1

    net start mysql

  5. 进行验证
